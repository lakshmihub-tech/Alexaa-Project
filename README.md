# Alexa Voice Assistant Project

This project is an implementation of a **voice-controlled assistant using  Alexa** and Python. It is designed to perform various tasks based on voice commands, including:

- Answering general questions
- Fetching weather updates
- Playing music
- Executing simple system commands
- Performing basic calculations

## Features

- Voice recognition and response
- Integration with APIs for real-time data
- Easy-to-use command interface
- Modular Python code for easy customization

## Technologies Used

- Python 3.x
- Amazon Alexa SDK
- SpeechRecognition and pyttsx3 libraries
- APIs (like weather, Wikipedia)
